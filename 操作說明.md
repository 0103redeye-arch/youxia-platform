# 遊俠平台 — 零基礎啟動說明

## 你需要的東西（先確認都裝好）

1. **Node.js** — 已裝好 ✓
2. **瀏覽器** — Chrome 或 Edge

---

## 第一步：建立資料庫（免費，5 分鐘）

1. 用瀏覽器開啟 **https://supabase.com**
2. 點「Start your project」→ 用 GitHub 或 Google 登入（沒有帳號就順便註冊）
3. 點「New project」，填：
   - Project name：`youxia`（隨便取）
   - Database Password：自己設一個密碼，**記下來**
   - Region：選「Northeast Asia (Tokyo)」
4. 等待約 1 分鐘建立完成
5. 進入專案後，點左側「Settings」→「Database」
6. 找到「Connection string」→「URI」，複製整串文字，長這樣：
   ```
   postgresql://postgres:[YOUR-PASSWORD]@db.xxxxx.supabase.co:5432/postgres
   ```
   把 `[YOUR-PASSWORD]` 換成你剛才設的密碼

---

## 第二步：設定環境變數

1. 打開 `C:\Users\林紓宇\Documents\youxia-platform\` 資料夾
2. 複製 `.env.example` 檔案，改名為 `.env.local`
3. 用記事本開啟 `.env.local`，填寫：

```
DATABASE_URL="貼上剛才從 Supabase 複製的連線字串"

NEXTAUTH_URL="http://localhost:3000"
AUTH_SECRET="隨便打一串很長的亂碼，例如：abcd1234efgh5678ijkl9012"
```

> 先只填這兩個就能跑，LINE/Google 登入之後再設。

---

## 第三步：建立資料表

1. 按 `Windows 鍵` + `R`，輸入 `cmd`，按 Enter
2. 複製貼上以下指令，按 Enter：

```
cd C:\Users\林紓宇\Documents\youxia-platform
```

3. 再貼這行，按 Enter：

```
"C:\Program Files\nodejs\npx.cmd" prisma db push
```

等待出現 `Your database is now in sync` 代表成功。

4. 匯入分類資料，執行：

```
"C:\Program Files\nodejs\npx.cmd" ts-node --compiler-options {\"module\":\"CommonJS\"} prisma/seed.ts
```

---

## 第四步：啟動網站

在同一個黑色視窗輸入：

```
"C:\Program Files\nodejs\npm.cmd" run dev
```

等出現 `Ready - started server on http://localhost:3000` 後，打開瀏覽器，網址輸入：

```
http://localhost:3000
```

看到遊俠平台首頁就成功了！

## 常見問題

**Q：關掉黑色視窗後網站就不見了？**
A：正常。要用的時候重新打開 cmd，cd 到資料夾，再跑 `npm run dev` 就好。

**Q：想讓別人也能用（不只是自己電腦）？**
A：需要部署到 Vercel。步驟：
1. 開啟 https://vercel.com，用 GitHub 登入
2. 把 `youxia-platform` 資料夾上傳到 GitHub
3. 在 Vercel 點「Import」→ 選這個專案
4. 把 `.env.local` 裡的內容填到 Vercel 的 Environment Variables
5. 按 Deploy，幾分鐘後就有一個公開網址

**Q：想開放 LINE / Google 登入？**
A：
- **Google**：去 https://console.cloud.google.com 建立 OAuth 憑證，取得 Client ID 和 Secret，填到 `.env.local`
- **LINE**：去 https://developers.line.biz 建立 Login Channel，取得 Channel ID 和 Secret
